# buyer_portal_mern
A buyer portal with authentication developed using MERN Stack



# Buyer Portal

## Setup
1) → npm install
2) → cd frontend : npm run dev
3) - cd backend → npm start

## Features
- Register/Login
- JWT Authentication
- Add/Remove favourites
- Home, Listing page
- Buyer's dashboard

## Flow
1. Register
2. Login
3. Add favourite property
4. View dashboard
    a. View favourite listings
    b. Remove porperties from favourite
